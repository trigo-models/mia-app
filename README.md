# MIA App - טופס רב שלבי

טופס רב שלבי בעברית עם תמיכה מלאה ב-RTL, מובייל פירסט ועיצוב שחור-לבן.

## תכונות

- ✅ תמיכה מלאה בעברית ו-RTL
- ✅ עיצוב מובייל פירסט
- ✅ 3 שלבים עם ולידציה
- ✅ עיצוב שחור-לבן מינימליסטי
- ✅ שימוש ב-SHADCN UI
- ✅ TypeScript
- ✅ Next.js 14

## התקנה

1. התקן את התלויות:
```bash
npm install
```

2. הפעל את השרת:
```bash
npm run dev
```

3. פתח את הדפדפן בכתובת: http://localhost:3000

## מבנה הפרויקט

- `app/` - דפי Next.js
- `components/` - רכיבי React
- `components/ui/` - רכיבי SHADCN UI
- `lib/` - פונקציות עזר

## שלבי הטופס

1. **פרטים אישיים** - שם, אימייל, טלפון, תאריך לידה
2. **פרטי התקשרות** - כתובת, עיר, אמצעי התקשרות מועדף
3. **העדפות ומידע נוסף** - תחומי עניין, ניסיון, זמינות

## עיצוב

הטופס מעוצב בסגנון מובייל פירסט עם:
- צבעים: שחור ולבן בלבד
- תמיכה מלאה ב-RTL
- רכיבי SHADCN UI
- עיצוב רספונסיבי
